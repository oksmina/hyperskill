Guess the number

In this task, you need to implement the algorithm of the well-known game “Guess the number”. The computer 'thinks' of a number and tells you the range in which the number is located. Of course, you could just output the number as if you had guessed it, but we'll do it another way.

Your task is to calculate the number of attempts (do you remember the Big O complexity?) it would take a binary search algorithm to guess the number.

Input: the first line is the number. The second line is the range (boundaries are included).

Output: number of attempts to guess the number.

At each iteration of the binary search algorithm, you need to check if the middle element is the one we're looking for.


Sample Input:
5
1 10

Sample Output:
1


Sample Input:
10
1 10

Sample Output:
4



Memory limit: 256 MB
Time limit: 15 seconds
