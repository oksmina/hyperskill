Min-max sorting

Let's say that a list is min-max sorted if the first element is the minimum, the second one is the maximum, the third is the second minimum and so on. Modify the described selection sort algorithm such that it performs a min-max sorting.

Input: one line containing integer numbers separated by spaces.

Output: min-max sorted list of the numbers.


Sample Input:
3 8 1 2 5 8

Sample Output:
1 8 2 8 3 5



Memory limit: 256 MB
Time limit: 15 seconds
