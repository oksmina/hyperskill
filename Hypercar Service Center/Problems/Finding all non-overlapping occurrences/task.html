Finding all non-overlapping occurrences

Modify the described algorithm in such way that it finds the indexes of all non-overlapping occurrences of a pattern in a text.

Input: two lines, the first of which represents the text, the other is the pattern.

Output: the indexes of all non-overlapping occurrences of the pattern in the text or $-1$ if no occurrences are found. For two overlapping substrings, consider the one with the least index as an occurrence. See the test example for details.


Sample Input:
aaaaa
aa

Sample Output:
0 2



Memory limit: 256 MB
Time limit: 15 seconds
