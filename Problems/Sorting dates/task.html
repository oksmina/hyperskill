Sorting dates

John works as a historian. He likes to collect notes about different historical events. Unfortunately, John usually makes his notes inconsequently. Help John to write a program that reorders his notes in the consistent order.

Input: one line containing dates separated by space. The date is represented by an integer number.

Output: the given dates sorting in descending order.

Modify the described version of the quick sort algorithm to solve the problem. Apply the 3-partition scheme. The idea of this approach is to divide the elements of a list into three groups instead of two: smaller, equal, and greater than the pivot. In this case, recursive calls of quicksort are needed to be performed only for the first and the third group. In particular, the algorithm works efficiently in the case when all elements of the input list are equal.


Sample Input:
1991 1984 2002 1998

Sample Output:
2002 1998 1991 1984



Memory limit: 256 MB
Time limit: 3 seconds
