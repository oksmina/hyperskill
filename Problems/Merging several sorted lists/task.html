Merging several sorted lists

In the basic implementation, the merge procedure takes two sorted lists as input and merges them into one sorted list. However, the procedure can be extended to merge several sorted lists. Your task here is to implement this extended version of the merge procedure.

Input: the first line contains an integer $n$, which is the number of sorted lists. Each of the following $n$ lines contains a list of integer numbers sorted in ascending order. The numbers are separated by space.

Output: a list of all input numbers sorted in ascending order.


Sample Input:
3
1 1 3 4 4 9
4 4 5 5 7
1 1 3 3

Sample Output:
1 1 1 1 3 3 3 4 4 4 4 5 5 7 9



Memory limit: 256 MB
Time limit: 2 seconds
