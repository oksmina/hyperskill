The number of merge operations

For lists of different sizes, the merge_sort algorithm performs a different number of merge operations. Write a program that for a given list counts the number of such operations.

Input: a list of integer numbers separated by spaces.

Output: the number of merge operations required to sort the list.


Sample Input:
3 1 2 4

Sample Output:
3



Memory limit: 256 MB
Time limit: 2 seconds
