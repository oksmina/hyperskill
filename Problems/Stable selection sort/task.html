Stable selection sort

The described version of selection sort is unstable: the initial order of equal elements might change after the sorting. However, the algorithm can be modified to be stable. Your task here is to implement a stable version of selection sort.

Input: the first line contains an integer $n$. Each of the following $n$ lines contains a string representing some name.

Output: the names sorted according to their lengths in ascending order (mind it doesn't mean alphabetically since you should only compare the lengths). Print each name on a separate line.

The sample demonstrates that swapping makes the selection sort unstable. Instead, you should insert the minimal element to the right position.


Sample Input:
6
Kate
Mike
Nick
Sam
Bob
Karl

Sample Output:
Sam
Bob
Kate
Mike
Nick
Karl



Memory limit: 256 MB
Time limit: 15 seconds
